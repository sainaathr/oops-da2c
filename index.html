Technology

Rock-solid
foundations

Built on modern, proven technologies — zero compromises on performance or reliability.

🔷

C++ (OOP)

Core engine in object-oriented C++ with classes, encapsulation, and clean architecture.

⚡

WebAssembly

Emscripten compiles C++ to WASM — near-native execution speed directly in the browser.

Vanilla JS

Zero framework dependencies. Pure HTML5, CSS3 and JS for maximum compatibility.

💾

File Persistence

Data written to inventory.txt via Emscripten's virtual filesystem — always persisted.
